7. Hand Gesture Based interactive Gaming Technology

07/24 - 08/24

Developed a Python application with a combination of OpenCV, MediaPipe, and pynput to create a hand gesture control system for a game. The script captures video frames from a webcam, processes them to detect hand landmarks using MediaPipe, and interprets gestures based on the distances between these landmarks. It then translates these gestures into keyboard actions using the pynput library. Specifically, gestures such as extending all fingers or closing them are mapped to in-game actions like jumping or stopping. The script also handles keypress durations to simulate actions like a long jump. The use of threading ensures that keypress actions can be performed asynchronously to prevent delays in gesture detection.
